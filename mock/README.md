# Como usar o Json-Server

1. Abra uma nova aba no terminal
2. Exetuce o comando:
``` 
yarn run json-server -w -p 333 ./mock/database.json  
```
ou 
```  
npm  run json-server -w -p 333 ./mock/database.json '''
```
